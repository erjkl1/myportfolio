## 技術選定の理由

昨今の UI,UX を最大化しつつ効率的に開発することができる<br>
フロントエンドモダン開発のうちデファクトスタンダードである技術を用いて開発していること<br>
<br>
自身の経験のある JS に近しい言語を選択して効率よく静的型付け言語を学びたかった<br>
自身のスキル向上のため習得難易度が高くても，広く活用されているかに着目し選択した
